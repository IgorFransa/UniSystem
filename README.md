# UniSystem — Gestão Universitária

O **UniSystem** é um **sistema web de gestão universitária**, desenvolvido como **projeto estudantil** por um grupo de **6 alunos**, com o objetivo de aplicar na prática conceitos de **engenharia de software, desenvolvimento web e orientação a objetos**.

> ⚠️ **Aviso importante**  
> Este projeto **não é um SaaS**, **não possui fins comerciais** e **não tem objetivo financeiro**.  
> O UniSystem foi criado **exclusivamente para fins acadêmicos e educacionais**.

---

## 🎓 Contexto Acadêmico

O projeto foi desenvolvido no contexto de atividades acadêmicas, com foco em:

- Desenvolvimento de sistemas orientados a objetos
- Organização e modelagem de sistemas complexos
- Estruturação de interfaces web (UI)
- Separação clara entre design, assets e implementação
- Trabalho em equipe e versionamento de código

O sistema simula um ambiente real de gestão universitária, **sem qualquer intenção de comercialização ou monetização**.

---

## 🧭 O que é o UniSystem?

O UniSystem representa um **painel de gestão acadêmica**, voltado principalmente para:

- Professores
- Administração acadêmica
- Controle e acompanhamento de informações educacionais

Funcionalidades conceituais representadas no sistema incluem:

- Gerenciamento de cursos
- Disciplinas
- Turmas
- Alunos
- Notas
- Frequências

Trata-se de um **sistema administrativo interno**, e não de um site institucional público.

- Professor

(outros papéis podem existir apenas como simulação acadêmica)

---

## 🖥️ Interface

A interface foi projetada no formato de **dashboard**, com:

- Sidebar de navegação
- Header contextual com identificação do usuário
- Área principal com indicadores e informações acadêmicas
- Uso de ícones e ilustrações em SVG
- Layout responsivo (conceitual)

---

## 🎨 Design e Assets

O projeto segue boas práticas de organização de design:

- **SVG** como formato padrão para ícones e ilustrações
- Separação entre:
  - design source (`design/`)
  - assets finais (`public/assets/`)
- Uso de CSS para controle de cores, estados e responsividade
